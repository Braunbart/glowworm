# Glowworm
Glowworm is a 3D LED mapping software.

## Project setup

### Install required dependencies
```
npm install
```

### Compile and hot-reload for development
On Linux:
```
npm run dev
```

On Windows:
```
npm run start:vue
npm run start:electron
```

### Compile and minify for production
```
npm run build
```

### Build the app into an executable
```
npm run dist
```
